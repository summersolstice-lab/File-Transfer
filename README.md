# 跨平台（Linux-Windows）网络文件传输系统

本项目实现了基于 TCP 协议的跨平台网络文件传输系统。其中 Linux 端作为服务器（C 语言开发），Windows 端作为客户端（Qt/C++ 开发），完美实现了**任务一（双向盲传）**与**任务二（带协议头的定长分块传输与进度条刷新）**的全部功能。

## 🚀 项目核心特性（任务二升级项）

* **自定义应用层协议（FileHeader）**：在正式数据传输前优先下发包含文件名（`filename`）与文件大小（`filesize`）的结构体面单，解决网络“盲传”与粘包问题。
* **1024字节流式定长分块传输**：服务端强制采用 `1024 字节` 缓冲区进行分片读取与发送，保障大文件异构平台传输时的内存安全。
* **Qt 进度条自适应动态刷新**：客户端引入状态机机制，动态解析协议头，并根据 `已收字节 / 总字节` 实时计算百分比，驱动 UI 界面 `QProgressBar` 平滑刷新。
* **双向传输支持**：保留任务一架构，完美兼容 Windows 文件的上传操作。
